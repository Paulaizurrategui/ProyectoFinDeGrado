# Hito 1 — Definición del proyecto

## Nombre provisional
TripBudget (Madrid → capitales europeas)

## Descripción
Aplicación Android que permite planificar un viaje en función de un destino y un presupuesto.
La app genera una propuesta de viaje con estimaciones de gasto (alojamiento, comidas y actividades)
y sugiere planes turísticos dentro del presupuesto. Además, los usuarios registrados pueden publicar
sus viajes para que otros los usen como ejemplo y pueden comentar publicaciones.

## Alcance inicial
- Origen fijo: Aeropuerto de Madrid (MAD)
- Destinos: (lista inicial de 8–12 capitales europeas)
- Generación de propuesta inicialmente con datos locales; posible ampliación con APIs externas.
