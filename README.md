# ProyectoFinDeGrado — App de viajes con presupuesto

Aplicación Android (Kotlin + Jetpack Compose) para planificar viajes desde Madrid a capitales europeas en función de un presupuesto.
Incluye generación de propuesta de viaje y una parte social para publicar y comentar viajes.

## Hitos
- Hito 1: docs/hito1 (definición, requisitos, modelo de datos, mockups)
- Hito 2: implementación 25%
- Hito 3: implementación 75% + borrador memoria
- Hito 4: entrega final

## Tecnologías previstas
- Kotlin
- Jetpack Compose
- MVVM
- Room (persistencia local)
- Firebase (Auth/Firestore/Storage) *(previsto)*
- APIs externas *(previsto como ampliación)*
